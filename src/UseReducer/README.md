# useReducer Hook
- khá giống với useState

## Cách dùng
- dùng useState với kiểu dữ liệu nguyên thủy
- dùng useReducer với trường hợp state phức tạp

```
useState
1. Init state: 0
2. Actions: Up(state + 1) / Down(state - 1)

```

```
useReducer
1. Init state: 0
2. Actions: Up(state + 1) / Down(state - 1)
3. Reducer
4. Dispatch (thuật ngữ dùng để kích hoạt 1 action)
```
